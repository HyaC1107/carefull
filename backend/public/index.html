Care-full 서버 기능 테스트

토큰 관리

저장 위치: 브라우저 localStorage

소셜 로그인

팝업에서 로그인 후 서버 콜백 응답(JSON)에서 token을 읽어 자동 저장합니다.

개발용 Mock 소셜 로그인

POST /api/user/dev-login

소셜 로그인 콜백 테스트

현재 구현된 것은 콜백 API뿐입니다. code와 state는 외부 OAuth 인증 후 받은 값을 직접 넣어 테스트합니다.

user.js 환자 등록 테스트

POST /api/user/register-patient

patient.js 환자 등록 테스트

POST /api/patient/register

patient.js 내 정보 조회 / 수정

device.js 기기 기능 테스트

POST /api/device/register, GET /api/device/me, DELETE /api/device/me

dashboard.js 대시보드 테스트

GET /api/dashboard

대시보드 API를 호출하면 요약 카드와 최근 데이터가 여기 표시됩니다.

medication.js 기능 테스트

GET /api/medication, GET /api/medication/search

schedule.js 기능 테스트

POST /api/schedule, GET /api/schedule, PUT /api/schedule/:id, DELETE /api/schedule/:id

activity.js 복약 로그 테스트

POST /api/log, GET /api/log, status: SUCCESS / FAILED / MISSED

face-data.js 얼굴 데이터 테스트

POST /api/face-data, GET /api/face-data

notification.js 알림 테스트

GET /api/notification, PATCH /api/notification/:id/read, PATCH /api/notification/read-all
